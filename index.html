Welcome to the Home Page!

This website is all about demonstrating the functionality of the navigation menu icon, also known as the "hamburger" icon, which only displays if the width of the current browser you're loading this website is less than or equal to 550px. Otherwise, you'll only see three available navigation menu options to choose from: Home, About, and Contact.

Feel free to adjust the width of the browser to see how these navigation menu icons change in response to different browser widths!
